# How to get an authorization token from fortnox?

For Omniboost to be able to connect to a fortnox account we need to have an authorization token. The steps below describe how to generate a new token in fortnox.

You need to be logged in to fortnox and have the 'manage users' rights to be able to do this. You also need to have the 'Integration' license enabled to be able to add new integrations.

## Go to manage users

## Click on the 'Add Integration' button

## Search for the Omniboost integration by code

Search for `GRRXwT5AtDYt`. Click on the 'Omniboost' integration and confirm it.

## Copy the authorization code and send it to Omniboost
